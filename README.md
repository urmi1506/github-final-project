# Simple Interest Calculator

## Description
A simple interest calculator that computes interest based on principal, rate, and time.

## Formula
Simple Interest (SI) = (Principal × Rate × Time) / 100

## Variables
- **P** = Principal amount
- **R** = Rate of interest (per year)
- **T** = Time (in years)

## Example
- Principal = 1000
- Rate = 5%
- Time = 2 years
- SI = (1000 × 5 × 2) / 100 = **100**
